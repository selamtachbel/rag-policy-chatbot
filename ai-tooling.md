AI assistance was used for:
- Understanding the project requirements
- Designing the Flask application structure
- Building the RAG workflow
- Connecting Sentence Transformers with ChromaDB
- Creating the web interface
- Debugging installation and runtime errors
- Writing documentation

I reviewed, tested, and customized the final project myself.
